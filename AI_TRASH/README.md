# AI_TRASH

Staging area for files that have been reviewed from AI_INBOX but are not currently needed in the project. Files here are **not automatically deleted** — they wait for a human review before being archived, moved, or permanently removed.

## How it works

1. Agent moves files here from AI_INBOX after reviewing them
2. Each file gets a brief note in the manifest below explaining why it was trashed
3. Periodically, Ryan and the agent review this folder together
4. For each file, the decision is: **archive** (move to reference/), **restore** (move back into project), or **delete** (permanent removal)
